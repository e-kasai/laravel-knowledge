### 説明
aタグをCSSでボタン風に見せる

### image
![[スクリーンショット 2025-07-30 094314 1.jpg]]

### code

```css
.header__link {
    /* 幅を固定してボタンらしい四角い見た目にする */
    width: 80px;
    /* ボタンの線を演出 */
    border: 1px solid #d9c6b5;
    /* ボタンの面を演出 */
    background-color: #f0f0f0;
    color: #d9c6b5;
    /* リンクの下線を消す */
    text-decoration: none;
}

```

```html
<a class="header__link" href="/login">login</a>

```
