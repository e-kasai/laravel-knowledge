### remとの違い
rem = 全体に対しての比率
em  = 親要素に対しての比率

### 使いどころ

| 用途 | なぜemが良いか | 解説 |
|------|----------------|------|
| ボタンやバッジ | 親文字サイズと連動して自然なバランスを保てる | ボタン内の文字と余白が比例して動く。親のfont-sizeが変わると、全体がきれいに縮小・拡大する。 |
| input / textarea | テキストと枠の比率が崩れない | 入力欄の文字サイズが変わってもpaddingが適切に保たれ、見た目の密度が均一になる。 |
| アイコン | フォントサイズにピッタリ揃う | width・heightを1emにすれば、テキストと常に同じ高さで揃えられる。 |
| 内側のpaddingやmargin | コンポーネント全体が一緒に縮む | 文字を小さくしたときに余白も自然に小さくなるため、詰まり感や空きすぎを防げる。 |
| セクション単位（ヘッダーなど） | 親単位でまとまってスケールできる | ヘッダーのfont-sizeを変えると、ロゴやナビなど全体が一体でスケールする。 |
