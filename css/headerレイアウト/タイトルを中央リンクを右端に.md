### 説明
タイトルを中央リンクを右端に配置するコード

### 解説
gridレイアウト

＜frとは＞
gridレイアウトの単位
コンテナ幅から パディングやマージン、固定幅（px など）を引いた「残り」を割合で分ける

ex: 4分割し真ん中を広めにとる
    grid-template-columns: 1fr 2fr 1fr;
    左カラム → 残りの 1/4、中央 → 2/4（1/2）、右 → 1/4 

ex2: 3等分
    /* 横方向 = 列は３分割で全部1frずつ, repeat(3, 1fr) = 1fr 1fr 1fr */
    grid-template-columns: repeat(3, 1fr);
    /* 縦方向 = 行は１行 */
    grid-template-rows: 1fr;


### よくあるトラブル

自動的に次の行に配置されてしまう

  原因：自動的に行が埋まっていると判断されて次の行になった  
  対処法：grid-rowも明示的に指定する。空いていればそこに入る
  例：grid-row:1;

### code
```css

.header {
    /* gridの適用 中身の要素を格子状に整列できるようにする */
    display: grid;
    /* 要素を縦中央に配置する */
    align-items: center;
    padding: 16px 0;
    /*ヘッダーの高さを固定 */
    height: 80px;
    border: 1px solid #e0dfde;
    /* 各セル内のインライン要素を左右中央に配置 */
    text-align: center;
    font-family: serif;

    /* 横方向 = 列は３分割で全部1frずつ, repeat(3, 1fr) = 1fr 1fr 1fr */
    grid-template-columns: repeat(3, 1fr);
    /* 縦方向 = 行は１行 */
    grid-template-rows: 1fr;
}

.header__heading {
    /* 通常の normal（400）よりもさらに軽いウェイト（300 相当など）を使用 */
    font-weight: lighter;
    font-size: 32px;
    /* ２番目の縦ラインから始まって隣のラインまでに配置 = ３列のうちの２列目を占有 */
    grid-column: 2;
}

.header__link {
    /* セル内の左右中央に配置(そのままだと左端に寄ってしまう為) */ 
    margin: 0 auto;
    /* 幅を固定してボタンらしい四角い見た目にする */
    width: 80px;
    /* ボタンの線を演出 */
    border: 1px solid #d9c6b5;
    /* ボタンの面を演出 */
    background-color: #f0f0f0;
    color: #d9c6b5;
    /* リンクの下線を消す */
    text-decoration: none;

    /* 3番目の縦ラインから始まって隣のラインまでに配置 = ３列のうちの3列目を占有 */
    grid-column: 3;
}

```
