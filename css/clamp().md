```php
width: clamp(50rem, 40vw, 90rem)
```


これを言い換えると：

「幅は基本的に画面の40%にしたいけど、
どんなに小さくても 50rem(500px) は確保して、
どんなに大きくても 90rem(900px) までにしてほしい」


### 挙動の例
ブラウザ幅が 2000px のとき
40vw = 800px (推奨値)

50rem = 約800px   (最小値)
90rem = 約1440px (最大値）

最小値と推奨値がほぼ同じ　→　幅は855pxくらいで止まってしまう

つまり、vw が思ったより小さいせいで、推奨値が最小値にぶつかって止まる


### 適用場面

フォントや余白など“伸び縮みしてほしいもの”に使う
```html
font-size: clamp(1rem, 2vw, 1.5rem);
margin: clamp(0.5rem, 2vw, 2rem);
```

レイアウトの幅は、まず固定＋`max-width`の方が確実
```html
width: 100%;
max-width: 900px;
```
