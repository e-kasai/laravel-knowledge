#### 前提

belongsTo() の第1引数（User::class）が「親となるモデル」、
第2引数（requested_by）が「外部キー（カラム名）」です。

この2つをつなぐ“橋の名前”が メソッド名（＝リレーション名） です。

#### 命名ルールの考え方

Eloquentは「メソッド名＝アクセス名」です。
たとえばこの定義：
```php
public function requester()
{
    return $this->belongsTo(User::class, 'requested_by');
}
```

はこう読みます。

“この `UpdateRequest` は、ひとりの `User` に属する。そのユーザーは `requester`と呼ぶ。”
