## コマンド

#### docker run = 下の３つをまとめて実行するコマンド

- docker pull　  ： 既成イメージのダウンロード　　　ex:docker pull nginx:latest
- docker create : コンテナの作成
- docker start   : コンテナの起動


## 既成イメージからコンテナ作成、起動までの例
```bash
//ex
docker run --name demo-nginx -d -p 8080:80 nginx


--name : 起動コンテナに名前を付けた
-d     : バックグラインド起動（ログを表示しないでCLIが使用できる）
-p     : ホストPCのポートとコンテナポートの導通（ ホストのPORT：コンテナのPORT ）
nginx  : 起動したいイメージの名前
```
