
---

#### やり方

** 1 コントローラーで型キャスト(変数の中身を、指定した型に変換する)を使ってtrue/falseフラグをたてる **
```php
// $profileの中身を true / false に変換
$canPurchase = (bool) $profile;
```

** 2 Bladeで3項演算子を使って条件分岐しボタンを切り替える **

```html
<button type="submit" {{ $canPurchase ? "" : "disabled" }}>購入する</button>
```

* 3項演算子
`条件 ? 真のとき : 偽のとき`

---

** 結果 **
- `$canPurchase = true` のとき:　 `""（空文字）`が出力されdisabled が出力されない　＝ ボタンは有効のまま
```html
//canPurchase = trueの場合
<button type="submit" >購入する</button>
```

- `$canPurchase = false` のとき: `disabled` が出力され、ボタンが無効になる
```html
//canPurchase = falseの場合
<button type="submit" disabled>購入する</button>
```
