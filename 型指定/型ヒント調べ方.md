
# Laravelで型を決める4ステップ

---

## STEP1: 返している関数やメソッドを確認
```php
return view('auth.register');
```
ここで使っているのは `view()` ヘルパー関数。

---

## STEP2: 定義にジャンプ
- `view` の上にカーソルを置く
- Ctrl + クリック（Macは Cmd + クリック）
- 関数の定義ファイルへジャンプ  

※ VSCodeだと、ジャンプ先が `vendor/laravel/framework/src/Illuminate/Foundation/helpers.php` などになる。

---

## STEP3: 戻り値型アノテーションを読む
ジャンプ先の `view()` 関数冒頭に以下のDocBlockが付いている。

```php
/**
 * Get the evaluated view contents for the given view.
 *
 * @param  string  $view
 * @param  array  $data
 * @param  array  $mergeData
 * @return \Illuminate\Contracts\View\View
 */
function view($view = null, $data = [], $mergeData = [])
```

- `@return \Illuminate\Contracts\View\View` が戻り値型。  
- このパスをコントローラーの先頭で `use Illuminate\Contracts\View\View;` と書けばOK。

---

## STEP4: 同じ手順で他の関数も調べる
例：`redirect()->intended()` の場合

1. `redirect` にカーソルを置いて定義へジャンプ
2. `redirect()` の戻り値は `Illuminate\Routing\Redirector`
3. `Redirector::intended()` にジャンプ
4. 戻り値に `Illuminate\Http\RedirectResponse` と書かれている

---

## 補足
- ホバー表示：カーソルを置くだけで戻り値型ヒントが出る（VSCodeのPHP Intelephenseなど有効化が必要）
- 公式ドキュメントのAPIリファレンス：https://laravel.com/api でクラスや関数名を検索可能

---

**まとめ**  
この手順を覚えておくと、自分で作ったメソッドにも正しい型を付けられるようになる。  
特に Laravel のヘルパーは DocBlock（`@return`）をそのまま信用して型指定すればOK。
