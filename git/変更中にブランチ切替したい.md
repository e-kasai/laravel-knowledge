### 説明
まだadd,commitしてないけど一旦ブランチを切り替えたい
今編集中の内容はあとでまた作業するから消えてほしくない

### code

```
# 例：READMEにWIPと追記
echo "WIP" >> README.md 
git status 
# 変更を一時退避
git stash push -m "wip: readme edit" 
# mainブランチに切り替え
git switch main 
# 元のブランチに戻る
git switch feature/a 
# 一時退避した変更を復元する（popはstashから削除もする、stashにのこしたければgit stash apply)
git stash pop 
```
