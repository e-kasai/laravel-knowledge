#### 手順

1. 環境を切り替える
`php artisan tinker --env=testing`

- 起動時に 環境を強制的に testing に切り替えるオプションで`.env.testing` を読み込む
- `APP_ENV  = testing` の状態（DBやキャッシュなども testing 用の設定に切り替わる)


2. 接続先とDBがテスト用になっているか確認
```bash
Deprecated: PHP Startup: Use of mbstring.internal_encoding is deprecated in Unknown on line 0
Psy Shell v0.11.2 (PHP 8.1.33 — cli) by Justin Hileman

//1つ目のテスト .env.testing が効いてるかチェック
>>> config('app.env')
=> "testing"     //結果：設定した接続先が返ってくればOK

//2つ目のテスト テストDBにつながってるか確認
>>> DB::connection()->getDatabaseName()
=> "app_test"    //結果：test用DBが返ってくればOK
>>>
```
