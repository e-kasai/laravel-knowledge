#### 手順

1. testファイルを作成

```bash
//phpコンテナ内
php artisan make:test HelloTest //testファイル名は一例
```
- make:test はデフォで Feature テンプレを作る

---

2. テストファイルを編集

//test内容によるので省略

---

3. test実行
```bash
//phpコンテナ内
php artisan config:clear   //先にキャッシュをクリアしておく
vendor/bin/phpunit tests/Feature/HelloTest.php
```
