1. config/service.phpにキーを登録
```php
// config/services.php にキーを登録
'stripe' => [
    'key' => env('STRIPE_KEY'),
    'secret' => env('STRIPE_SECRET'),
    'webhook_secret' => env('STRIPE_WEBHOOK_SECRET'),
],
```

---

2. CSRF除外
```php
// CSRF除外（app/Http/Middleware/VerifyCsrfToken.php）
protected $except = ['stripe/webhook'];
```

なぜ除外するか

- CSRFは「"ブラウザ利用者の"意図しないリクエスト」を防ぐ仕組み。
- Webhookは「外部サービス（Stripe）からのサーバー間通知」だからCSRFの守備範囲外
- だから VerifyCsrfToken では 例外扱いにして素通りさせる
- CSRF代わりにStripeの署名チェック(whsec_xxx がそのための鍵で、これを .env に入れる)で守備
